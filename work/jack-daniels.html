About

Brief

I wanted to create a portfolio for a fictional web developer based in London. I like playing around with responsiveness, and for this I had the idea to keep as much content as possible, even for the mobile display.

Result

I went for a clean, modern design, showing a full portrait photo with useful information on the right-hand side. For the weather, I used an API which updates the weather icon and temperature hourly, and also grabs the location that you write (in this case, London). For the portfolio, I designed a full-screen slideshow, displaying high-res images with tags at the bottom. Regarding the mobile design, rather than crowding the view, I decided to put the portrait image in the background, faded.
