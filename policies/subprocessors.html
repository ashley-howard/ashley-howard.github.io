Subprocessors for ashley.how

(N.B. "we", "us", and "ours", are used to refer to the site owner Ashley Howard and his sole-trader web design business)

We use third party subprocessors, such as cloud computing providers and customer support software, to run websites for particular services. We establish GDPR-compliant data processing agreements with each subprocessor, extending GDPR safeguards everywhere personal data is processed.

The following is a list of personal data subprocessors we use. It may vary according to the client's needs. The following subprocessors are all located in the United States:

  • Amazon Web Services. Cloud services provider.
  • Google Cloud Platform. Cloud services provider.
  • GoDaddy. Domain and Web hosting services provider.
  • Github. Web hosting services provider.
  • Mailchimp. Email newsletter service.
  • Paypal. Payment transfer service.
  • Shopify. E-commerce platform.
  • WooCommerce. E-commerce platform.
